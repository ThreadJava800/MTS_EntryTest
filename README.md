# Перекладыватель YAML-ов

Программа принимает на вход путь до некоторого .gitlab-ci.yml файла, парсит его и возвращает структуру Kotlin-объектов.

Программа ожидает на вход файл следующего формата:

- Комментарии должны начинаться с символа "#"
- В качестве отступов допускаются лишь пробелы (не табы!)
- У блоков одного уровня должны быть одинаковые отступы
- В качествые ключей допускаются лишь слова (в которых также могут быть символы "-" и "_")
- В качестве значений допускаются слова и строки в кавычках

- Обязателен список stages
- Внутри каждого блока верхнего уровня должно быть поле image
  image - либо строка, либо объект с полем name

Запуск:
```bash
./gradlew run --args="examples/.gitlab-ci.yml"
```
